## ðŸ§  Behavioral Psychology Foundation

### The Science of Referral Motivation

People don't refer others just for money. Research shows the most effective referral programs tap into **six core psychological drivers**:

1. **Reciprocity** - "They gave me value, I want to give back"
2. **Social Proof** - "Everyone I know is doing this"
3. **Loss Aversion** - "I can't afford to miss out on this"
4. **Status & Recognition** - "I want to be seen as successful/helpful"
5. **Progress & Achievement** - "I'm building toward something meaningful"
6. **Altruism** - "I genuinely want to help people I care about"

**Key Insight**: Financial incentives alone create transactional relationships. Emotional drivers create passionate advocates.

**Scenario**: Customer uses coach referral code (15%) + loyalty points (CHF 100) + customer referral code (10%)

- **Total discount**: CHF 30 + CHF 100 + CHF 20 = CHF 150 (75% off!)
- **Result**: Unprofitable

### Solution: Smart Discount Rules

**Rule 1: Referral Code Mutual Exclusivity**
- Coach referral code **OR** customer referral code
- Not both
- UI: "You've applied a coach referral code. Customer referral codes cannot be combined."

**Rule 2: Loyalty Points Always Stack**
- Loyalty points can combine with ONE referral code
- Why: Points were earned through past purchases - they deserve to use them

- First-time customers: Max 35% off total order
